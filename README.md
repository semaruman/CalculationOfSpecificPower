  Далее мой заказчик сказал мне, что последние 4 параметра можно получить из таблицы по первому параметру и объяснил как и попросил сделать приложение, которое:
    <ul>
      <li>Должно запрашивать количество потребителей (первый параметр)</li>
      <li>Должно давать выбор типа потребителя из списка (на данном примере типов потребителей - 4)</li>
      <li>Должно расчитывать момент и ток</li>
      <li>Должно быть настольным с удобным интерфейсом</li>
